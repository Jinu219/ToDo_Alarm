## 24-1 WAP 앱스터디 토이프로젝트 : ToDo_Alarm

### 프로젝트 소개 
- 프로젝트의 이름은 “To Do Alarm”이며. 투두 리스트와, 알람 기능이 합쳐진 앱입니다.
- 아이폰의 기본 앱인 **"미리 알림"** 이라는 앱을 클론 코딩 하여 프로젝트를 진행하였습니다.
